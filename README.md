# WildfireWatch AI App


This app was created automatically by WildfireWatch AI.
It's a Vite+React app that communicates with the WildfireWatch AI API.

## Running the app

```bash
npm install
npm run dev
```

## Building the app

```bash
npm run build
```

For more information and support, please contact WildfireWatch AI support at app@wildfirewatch.com.
